# Instructor Guide

## Suggested Exercises
- Run the demo in fixture mode and recover a key
- Change leakage bits and observe recovery/failure
- Compare RFC6979 mode to biased/fixed nonce modes
- Analyze a historical case study from `src/data/case-studies.ts`
- Export/import demo evidence and discuss reproducibility

## Teaching Notes
- Emphasize educational vs. cryptographic claims
- Discuss real-world mitigations and limitations
- Use the threat model and limitations docs for discussion prompts
